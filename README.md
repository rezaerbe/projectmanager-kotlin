## The Complete Android 10 & Kotlin Development Masterclass

This is course about Learn Android 10 App Development From Beginner to Advanced Developer. Learn to Code Kotlin While Building Android Q Apps.

## Tools

* Android Studio
* Kotlin Language

## Dependencies

* Material
* Firebase Auth
* Firebase Firestore
* Firebase Storage
* Firebase Mesagging
* Circle Image View
* Glide
